#### API란?
- Application Programming Interface
- 한 프로그램에서 다른 프로그램으로 데이터를 주고받기 위한 방법 
- 사용자가 서비스를 이용할 수 있게 미리 만들어 둔 메뉴판 
- API가 가져야할 내용
  1. 요청방식(데이터 요청? 데이터 응답?)
  2. 어떤 자료를 요청 (endpoint(url))
  3. 자료요청에 필요한 추가 정보 (파라미터)
- web의 경우 REST API를 따르는 것을 추천
